# ABI Consulting Website

Static site for ABI Consulting, hosted on GitHub Pages.

## Deploy
Push changes to the `main` branch of `jagrmeister.github.io`. GitHub Pages redeploys automatically.

## Structure
- `index.html` – main site
- `assets/` – favicon and **individual certification badges**
- `.nojekyll` – disables Jekyll processing

## Certification badges
Place your files in `assets/` with exactly these names:
- `GCIA.png`, `GWAPT.png`, `GCED.png`, `GEVA.png`
- `CCSP.png`, `SSCP.webp`
- `Aplus.webp`, `Proj_plus.png`, `networkplus-logo.png`, `Pentest_plus.png`, `CySA.png`
- `ECES.avif`, `ITIL.png`

Logos are display-only and scaled consistently by CSS.
